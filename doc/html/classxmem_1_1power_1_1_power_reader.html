Detailed Description

An abstract base class for measuring power from an arbitrary source. This class is runnable using a worker thread.

Constructor.

Parameters
sampling_periodThe time between power samples in seconds.
power_unitsThe power units for each sample in watts.
nameThe human-friendly name of this object.
cpu_affinityThe logical CPU to be used by the thread calling this object's run() method. If negative, any CPU is OK (no affinity).
